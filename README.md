# ☕ AI Customer Support Copilot

An AI-powered Customer Support Copilot developed for **Kopi Santai Café** using **Google Gemini API**.

---

## 📌 Project Overview

This project aims to build an intelligent customer support assistant that can:

- Understand natural language from customers.
- Recommend menu combinations based on customer preferences and budget.
- Answer café-related questions only.
- Reject questions outside the café domain (Domain Guard).

---

## ✨ Main Features

- 🤖 Natural Language Understanding (NLU)
- ☕ Menu Recommendation
- 💰 Budget-Based Recommendation
- 🛡️ Domain Guard
- 💬 Real-Time Chat Support
